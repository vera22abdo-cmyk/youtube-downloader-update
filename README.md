# AP-SOLOX4 YouTube Audio/Video Downloader

---

## 📋 وصف التطبيق / Application Description

**العربية:**
تطبيق احترافي لتنزيل الفيديوهات والموسيقى من YouTube يدعم تنزيل الملفات بصيغ MP3 و MP4 بجودة عالية.

**English:**
A professional application for downloading videos and music from YouTube, supporting MP3 and MP4 formats with high quality.

---

## ✨ الميزات / Features

### 🎵 تنزيل الصوتيات (MP3) / Audio Download (MP3)

**العربية:**
- دعم جودة 320kbps (عالية)
- دعم جودة 192kbps (متوسطة)
- دعم جودة 128kbps (صغيرة)
- تضمين الصورة المصغرة (Thumbnail)
- منع التنزيل المكرر داخل نفس المجلد

**English:**
- Support for 320kbps quality (High)
- Support for 192kbps quality (Standard)
- Support for 128kbps quality (Low)
- Embed thumbnail
- Prevent duplicate downloads within the same folder

### 🎬 تنزيل الفيديوهات (MP4) / Video Download (MP4)

**العربية:**
- جودة عالية (Best Video + Best Audio)
- دعم التنزيل من القوائم التشغيل
- خيار تنزيل فيديو واحد فقط

**English:**
- High quality (Best Video + Best Audio)
- Support for playlist downloads
- Option to download single video only

### 🎨 واجهة المستخدم / User Interface

**العربية:**
- تصميم داكن احترافي (Premium Dark Theme)
- شريط تقدم ديناميكي متزامن مع التنزيل
- سجل حالة ملون (أخضر، أحمر، برتقالي، سماوي)

**English:**
- Professional dark theme design
- Dynamic progress bar synchronized with download
- Colored status log (green, red, orange, cyan)

### 🔄 التحديث التلقائي / Auto Update

**العربية:**
- فحص التحديثات تلقائياً
- تحديث yt-dlp.exe و FFmpeg.exe تلقائياً
- تحديث التطبيق نفسه

**English:**
- Automatic update checking
- Automatic update of yt-dlp.exe and FFmpeg.exe
- Self-update for the application

## 🚀 التثبيت / Installation

### المتطلبات / Requirements

**العربية:**
- Windows 10 أو أحدث
- اتصال بالإنترنت (للتحديثات والتنزيل)

**English:**
- Windows 10 or later
- Internet connection (for updates and downloads)

### التشغيل / Running

**العربية:**
1. حمل الملف `AP-SOLOX4.exe`
2. شغله مباشرة (لا يحتاج لتثبيت)
3. التطبيق سيقوم بتحميل yt-dlp.exe و FFmpeg.exe تلقائياً

**English:**
1. Download the `AP-SOLOX4.exe` file
2. Run it directly (no installation required)
3. The application will automatically download yt-dlp.exe and FFmpeg.exe

## 📖 الاستخدام / Usage

### تنزيل فيديو/موسيقى / Download Video/Music

**العربية:**
1. انسخ رابط YouTube
2. الصقه في حقل URL
3. اختر الصيغة (MP3 أو MP4)
4. اختر الجودة
5. اضغط "START"

**English:**
1. Copy the YouTube URL
2. Paste it in the URL field
3. Choose the format (MP3 or MP4)
4. Choose the quality
5. Press "START"

### إلغاء التنزيل / Cancel Download

**العربية:**
- اضغط "CANCEL & CLEAN" لإلغاء وحذف الملفات غير المكتملة

**English:**
- Press "CANCEL & CLEAN" to cancel and delete incomplete files

### تغيير مجلد التنزيل / Change Download Folder

**العربية:**
- اضغط "Change" لاختيار مجلد جديد
- اضغط "Open" لفتح مجلد التنزيل الحالي

**English:**
- Press "Change" to choose a new folder
- Press "Open" to open the current download folder

### تحديث الأدوات / Update Tools

**العربية:**
- اضغط "🔄 Update" لتحديث yt-dlp و FFmpeg يدوياً

**English:**
- Press "🔄 Update" to manually update yt-dlp and FFmpeg
